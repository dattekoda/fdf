> 操作方法   
【視点移動】   
`↑`: 上   
`↓`: 下   
`→`: 右   
`←`: 左   
【回転】   
`w`: 縦方向上側   
`s`: 縦方向下側   
`a`: 横方向左側   
`d`: 横方向右側   
【ズーム】   
`+`: 寄り   
`-`: 引き   
【リセット】
`x`: x軸   
`y`: y軸   
`z`: z軸   
`i`: 初期のリセット   
【伸ばす】   
`>`: 上方向   
`<`: 下方向
